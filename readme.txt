IMPORTANTE

Este proyecto ha sido ejecutado en Android Studio, mediante un emulador Pixel4 API27 6GB de RAM, si se va a utilizar otro emulador procurar que tenga como mínimo 4GB de RAM para que pueda ejecutarse correctamente.
Además, configurar el IDE con el compileSDK35 mediante el gradle.
